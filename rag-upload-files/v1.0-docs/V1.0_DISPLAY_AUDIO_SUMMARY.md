# Ghetto Blaster v1.0 - Display & Audio Setup Summary

**Quick Reference for Ghetto Blaster Display & Audio Configuration**

## Display: 1280x400 Landscape

- **Hardware:** Waveshare 7.9" HDMI (400x1280 native, rotated to 1280x400)
- **Boot Config:** `/boot/firmware/config.txt` (KMS enabled, HDMI forced)
- **X11 Rotation:** `/home/andre/.xinitrc` (dynamic HDMI detection, rotate left)
- **Applications:** PeppyMeter (VU meters) or Chromium (moOde UI/Wizard)

## Audio: HiFiBerry AMP100

- **Hardware:** HiFiBerry AMP100 (2x50W, I2S)
- **Routing:** MPD → _audioout → peppy → _peppyout → camilladsp → DAC
- **Config:** `/usr/share/camilladsp/working_config.yml`
- **Volume:** **MAX 75% (191/255)** - Current: 79% (202) - **REDUCE IMMEDIATELY**

## Key Files
