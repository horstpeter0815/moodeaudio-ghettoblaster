## Audio Configuration Files (KEEP)

### CamillaDSP
- `/usr/share/camilladsp/working_config.yml` - Main CamillaDSP configuration
  - Contains: Device settings, filter pipeline, EQ bands
  - **Critical:** Schema uses `channels:` (not `channel:`)

### ALSA Configuration
- `/etc/alsa/conf.d/camilladsp.conf` - ALSA CamillaDSP plugin
  - Contains: Plugin configuration, sample rates, state file path

- `/etc/alsa/conf.d/_peppyout.conf` - PeppyMeter output routing
  - Contains: Routes PeppyMeter output to CamillaDSP

- `/etc/alsa/conf.d/_audioout.conf` - Main audio output routing
  - Contains: Routes audio to PeppyMeter for VU meters

### Backup Files (KEEP LATEST ONLY)

- `/etc/alsa/conf.d/_peppyout.conf.bak_20260106_175807` - Backup from peppy fix
