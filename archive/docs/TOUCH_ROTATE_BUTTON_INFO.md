## TOUCH ROTATE BUTTON

The display has a **rotate touch button** on the back.

This button typically:
- **Rotates touchscreen calibration** (not display rotation)
- Changes how touch coordinates are interpreted
- May cycle through rotation modes (0°, 90°, 180°, 270°)

---

## CURRENT TOUCHSCREEN CONFIGURATION

- **Transformation Matrix:** Currently set to 270° rotation (0 -1 1 1 0 0 0 0 1)
- **Display:** Landscape (1280x400)
- **Touch:** Should match display orientation

---

## WHEN TO USE THE BUTTON

Use the rotate touch button if:
- Touch coordinates don't match where you touch
- Touch works but is rotated incorrectly
- After changing display orientation

---

## TESTING

Before using the button:
1. Test current touch calibration
2. Verify if touch coordinates match display
3. If misaligned, try the rotate touch button
