# Peppy Meter Research

## What is Peppy Meter?
Peppy Meter is a music visualizer that displays audio waveforms and visualizations on a display, originally designed for Raspberry Pi with small displays.

## Key Features
- Real-time audio visualization
- Multiple visualization modes
- Configurable for different display sizes
- Python-based
- Works with ALSA audio

## Installation Methods

### Method 1: Direct Installation
- Clone from GitHub
- Install Python dependencies
- Configure for display
- Run as service

### Method 2: Moode Audio Plugin
- Check if Moode has Peppy Meter plugin
- Install via Moode UI or manually
- Configure in Moode settings

## Configuration for 1280x400
- Display resolution: 1280x400
- Landscape orientation
- Touchscreen support (if needed)
- Integration with Moode Audio player

## Requirements
- Python 3
- Pygame or similar graphics library
- ALSA audio access
- Display configured (already done)
