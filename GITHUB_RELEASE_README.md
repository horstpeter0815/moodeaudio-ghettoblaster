## Quick Download & Burn

1. **Download** the latest release: `moode-10.2-working-YYYYMMDD.img.zip`
2. **Extract:** `unzip moode-10.2-working-*.img.zip`
3. **Burn** to SD card using Balena Etcher or dd
4. **Boot** Raspberry Pi 5
5. **Access:** http://moode.local
