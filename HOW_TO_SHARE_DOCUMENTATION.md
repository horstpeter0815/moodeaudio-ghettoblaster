# How to Share HARDWARE_CONFIGURATION.md with Colleague

## File Location
```
~/moodeaudio-cursor/HARDWARE_CONFIGURATION.md
```

## Sharing Options

### Option 1: Email (Easiest)
1. Open the file
2. Copy all content (Cmd+A, Cmd+C)
3. Paste into email
4. Send to colleague

Or attach file:
1. Attach `HARDWARE_CONFIGURATION.md` to email
2. Send to colleague

### Option 2: File Sharing Service
**Dropbox/Google Drive/iCloud:**
1. Upload `HARDWARE_CONFIGURATION.md` to cloud storage
2. Share link with colleague
3. Colleague downloads file

**Quick upload command:**
```bash
# If you have cloud sync folder
cp ~/moodeaudio-cursor/HARDWARE_CONFIGURATION.md ~/Dropbox/
# Or
cp ~/moodeaudio-cursor/HARDWARE_CONFIGURATION.md ~/Google\ Drive/
```

### Option 3: GitHub (If using Git)
1. Commit file to repository
2. Push to GitHub
3. Share repository link or create Gist
4. Colleague can view/download

**Commands:**
```bash
cd ~/moodeaudio-cursor
git add HARDWARE_CONFIGURATION.md
git commit -m "Add hardware configuration documentation"
git push
# Share repository URL
```

### Option 4: Copy to USB/External Drive
1. Copy file to USB drive
2. Give USB to colleague

**Command:**
```bash
cp ~/moodeaudio-cursor/HARDWARE_CONFIGURATION.md /Volumes/USB_DRIVE/
```

### Option 5: Direct Copy (Same Network)
If colleague is on same network:
1. Enable file sharing
2. Share file via network

### Option 6: Print/Save as PDF
1. Open file in text editor or markdown viewer
2. Print to PDF
3. Email PDF or share file

**Command (macOS):**
```bash
# Convert to PDF using textutil or print command
textutil -convert pdf ~/moodeaudio-cursor/HARDWARE_CONFIGURATION.md -output ~/Desktop/HARDWARE_CONFIGURATION.pdf
```

### Option 7: Paste in Shared Document
1. Copy content from file
2. Paste into:
   - Google Docs
   - Microsoft OneNote
   - Shared wiki/documentation system
   - Slack/Teams message

---

## Recommended Method

**For quick sharing:** Email (copy/paste content)  
**For permanent access:** GitHub or cloud storage (Dropbox/Drive)  
**For formatted version:** Convert to PDF and email
