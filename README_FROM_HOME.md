# Network Fix Commands - From Home Directory

All commands work from `~/` (home directory).

## Apply Minimal Network Fix

```bash
cd ~/moodeaudio-cursor && sudo bash MINIMAL_NETWORK_FIX.sh
```

Or use the helper script:
```bash
bash ~/moodeaudio-cursor/APPLY_MINIMAL_FIX_FROM_HOME.sh
```

## Test Mac Ethernet

```bash
bash ~/moodeaudio-cursor/TEST_MAC_ETHERNET.sh
```

## Check Fix Status

```bash
bash ~/moodeaudio-cursor/CHECK_TIMING_FIX_STATUS.sh
```

## All Commands From Home Directory

All scripts can be run from `~/` without needing to `cd` first (they handle it internally).
